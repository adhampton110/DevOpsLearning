Welcome

DevOpsLearning is a repository where I document my journey of learning various DevOps practices and tools to improve the speed and quality of software development, deployment, and maintenance. The goal is to develop a minimal application (this webpage) that utilizes various services commonly used in the DevOps lifecycle. I will release markdown notes of my findings and plans, so others can benefit from it and enhance the resources.

This will not cover all things "DevOps" but it will cover the areas that I feel will benefit my learning and understanding overall.

Course of Action

To achieve my goal, I plan to gain knowledge on the following concepts:

  • Learn a programming language
  • Infrastructure as Code
  • Continuous Integration
  • Continuous Delivery
  • Monitoring software and infrastructure
  • Cloud providers

The DevOps stack I plan to incorporate for this repository includes:

  • HTML, CSS, JS, and Node JS for full-stack development
  • GitHub for version control
  • Docker for containerization
  • Jenkins for CI/CD
  • Terraform for infrastructure monitoring
  • Jaegar for application monitoring
  • AWS ECR for application deployment
